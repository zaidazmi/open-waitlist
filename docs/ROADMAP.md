# Roadmap

Open Waitlist starts intentionally small. These are the next improvements that would make it stronger for production launches.

## Near Term

- Optional Turnstile/CAPTCHA adapter
- Disposable-email protection
- Admin export endpoint
- Tests for API routes

## Later

- React package for drop-in forms
- Embeddable script widget
- Basic admin dashboard
- Email template customization UI
- Multiple waitlists/projects in one deployment
- Provider adapters for Postgres, Neon, and plain SQL
