# Changelog

All notable changes to Open Waitlist will be documented in this file.

## 0.1.0 - Initial Public Release

- Added the standalone Next.js waitlist app.
- Added Supabase-backed signup storage and referral tracking.
- Added Resend-powered double opt-in verification emails.
- Added backend-only mode for custom frontends.
- Added signup rate limiting, email resend limiting, and honeypot/timing bot checks.
- Added dynamic rank queries and atomic verification/referral crediting.
- Added verification resend behavior for unverified duplicate signups.
- Added setup docs, API docs, deployment docs, and contribution templates.
